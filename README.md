This library is a standalone observability layer for agent systems. It wraps important units of work, like benchmark runs, tool calls, and LLM calls, inside trace spans. Those spans carry metadata such as model, backend, duration, and exit status. The library can emit standard OpenTelemetry traces, but it also writes simple JSONL records so it remains useful in fully isolated environments. The main design idea is to make agent execution inspectable without coupling observability to a specific harness or vendor backend.
